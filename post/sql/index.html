SELECT FROM

  • DISTINCT 去重
  • CASE WHEN END 给数据分组打标签
  • 窗口函数

WHERE

  • <=> between
  • (not)in
  • and or(and优先于or执行)
  • like（%匹配字符%）

ORDER BY

  • 默认升序ASC
  • 降序关键字DESC

GROUP BY

  • 聚合函数
    • COUNT()
    • SUM()
    • AVG()
    • MAX()
    • MIN()
  • 分组前过滤用WHERE，分组后过滤用HAVING

函数

字符串

  • LEFT()
  • LOCATE()
  • LENGTH()
  • SUBSTR()

时间

  • DATEDIFF()
  • DATE_SUB()
  • DATEFORMATE()

ROW_NUMBER() OVER (PARTITION BY ORDER BY )

目的

分组后返回多行，可进行组内排序等操作

窗口函数

聚合函数

SUM();COUNT();AVG();MIN();MAX()
默认聚合第一行到当前行累计的值
对ROWS操作：

  • n PRECEDING 向前n行
  • n FOLLOWING 向后n行
  • CURRENT ROW 当前行

组内排名

  • ROW_NUMBER() 忽略重复值
  • RANK() 重复值累计
  • DENSE_RANK() 重复值不累计

返回组内某一行的值

  • FIRST_VALUE()
    返回窗口内第一行的值

  • LAST_VALUE()
    返回窗口内最后一行的值

  • LAG(A,n,null)
    返回A列当前行之前第n行的值，若不存在则返回null

  • LEAD(A,n,null)
    返回A列当前行之后的第n行的值，若不存在则返回null

    SQL执行顺序

    1. FROM+JOIN
      首先把表准备好
    2. WHERE
      初步过滤
    3. GROUP BY
      分组
    4. HAVING
      分组后过滤
    5. SELECT
      把要查的内容找到
    6. ORDER BY
      按顺序排列已查到内容
    7. LIMIT
      截取需要的内容
